TLS Certificate Setup

This is a one-time configuration step. Vork Relay will request a free TLS certificate from Let's Encrypt for your domain using the HTTP-01 challenge. Once issued, the server restarts in HTTPS mode. Certificates renew automatically 30 days before expiry — no further action is needed.

Prerequisites

  • This server's domain name (e.g. relay.example.com) has a DNS A record pointing to this machine's public IP.
  • Port 80 is open on this machine's firewall and reachable from the internet.
  • Nothing else is bound to port 80 on this host.
Used by Let's Encrypt for expiry notifications.
